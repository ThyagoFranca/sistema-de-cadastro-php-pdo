Esse é um sistema de cadastro de usuários e login completo, a ideia é que através desses arquivos você tenha uma base para arquitetar o seu projeto.

Adicionei apenas uma variavel CPF que acredito ser muito importante para qualquer sistema de cadastro e login de usuários.

Através dessa única variavel você conseguira desenvolver a sua lógina para os demais campos como por exemplo;

$nome
$email
$phone

etc...

Usando a mesma lógica.

E o mais importante é que podemos tratar erros usando apenas o PHP do tipo como CPF INVÁLIDO ou CPF JÁ EXISTE dando assim informações ao usuário do que deve ser feito.

Nada impede você de usar JavaScript para tratar os erros. 

Espero que isso ajude aos novatos que assim como eu tive muitas dúvidas no início da minha carreira como dev.
